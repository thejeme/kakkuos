# Claude Context

Use `AGENTS.md` as the project context and engineering guide for this repository. Use `TWEAKING.md` when helping normal users customize or troubleshoot a running KakkuOS desktop.

KakkuOS is a CachyOS-based desktop layer built around niri, DankMaterialShell, Ghostty, fish, Starship, Zen Browser, PipeWire, and curated package/default sets. Keep changes consistent with the layered model described in `AGENTS.md`.
